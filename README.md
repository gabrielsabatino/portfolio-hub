## 🎯 Projetos em Destaque

### 📊 [Análise Exploratória de Vendas](./projetos/analise-vendas/)

Análise de dados de vendas usando **Pandas** e **Matplotlib**. Identifica categorias mais lucrativas, top produtos e ticket médio através de estatísticas e visualizações.

**Tecnologias:** `Python` `Pandas` `NumPy` `Matplotlib`

---

### 🌸 [Classificador de Espécies Iris](./projetos/classificador-iris/)

Modelo de **Machine Learning supervisionado** que classifica flores Iris em três espécies usando o algoritmo K-Nearest Neighbors do **scikit-learn**, atingindo acurácia superior a 95%.

**Tecnologias:** `Python` `scikit-learn` `NumPy` `Matplotlib`

## 🚀 Como Usar Este Repositório

### Clonar localmente

```bash
git clone https://github.com/gabrielsabatino/portfolio-hub.git
cd portfolio-hub
```

### Executar um projeto

Cada pasta dentro de `projetos/` contém seu próprio `README.md` com instruções específicas.

```bash
cd projetos/analise-vendas
pip install pandas numpy matplotlib
python analise_vendas.py
```
